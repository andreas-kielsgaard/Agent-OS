# Agent Attention System

The agent attention system is the core routing layer that helps an implementation agent decide what to load and how to work for a given task.

It is expected to cover:

- task-mode selection;
- behavior selection for structural maintenance work;
- lens selection for focused reasoning;
- skill selection for repeatable workflows;
- tool and evidence routing without treating tool output as authority.

This directory is intentionally a placeholder in the foundation branch. Reusable files should be extracted from the Field Platform reference installation only after the core contract and project-specific boundaries are clear.
